# How to use

### clone git

```
> git clone https://github.com/hailt88/shopdevtest.git <your-project-name>
> cd your-project
git init
git add .
git commit -m "message"
git remote add origin <git url>
git push -f origin main
-----------
> npm i
> npm run dev
```
